# Credit-Fraud

Project ini merupakan studi kasus yang berfokus pada Credit Fraud yang datasetnya diambil dari Kaggle. Metode yang digunakan terdiri dari
1. Data Understanding
2. Data Peprocessing
  * Feature Selection
  * Handling Skew
  * Data Preparation 
3. Modeling menggunakan Random Forest dengan parameter
  * n_estimators
  * min_leaf_split
  * criterion
  * Max Depth

Proses evaluasi yang digunakan ialah akurasi, recall, precision dan F1-Score dengan metode Weighted Evalution

4. Deployment yang berbasis API dengan FAST API dan Docker
